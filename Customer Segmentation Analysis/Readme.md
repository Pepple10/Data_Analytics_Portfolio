**Customer Segmentation Analysis**



**Objective**



This project analyzes customer data to identify key segments based on income, spending behavior, and activity level. The goal is to support targeted marketing and improve customer retention.



**Methodology**

* Cleaned and prepared a messy dataset in Excel
* Created derived features such as income level, spending group, and customer status
* Performed segmentation analysis using MySQL
* Built an interactive dashboard in Power BI



**Key Insights**

* High-income customers contribute significantly but form a smaller segment
* A large portion of customers fall into low and medium spending categories
* A noticeable share of customers are inactive, indicating retention issues
* Customer distribution is concentrated in a few key cities



**Recommendations**

* Focus marketing efforts on high-income, high-spending customers
* Develop re-engagement campaigns for inactive customers
* Target high-growth cities to expand the customer base
* Create tailored offers for different income segments



**Tools Used**



* Excel
* MySQL
* Power BI
